Melita
======

R scripts used for analysis of Melita dataset

Data description:

6 biological replicates each

3 treatments:
- early leaf removal (PFLR)
- variation (grape coloring) leaf removal (VLR)
- control (no leaf removal) (NLR)


2 different expositions:
- north (N)
- south (S)


2 different sites:
- slovenia (SLO)
- italia (ITA)
